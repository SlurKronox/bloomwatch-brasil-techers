# Testing

## Estratégia
- Unit tests para regras de previsão
- Unit tests para cliente API
- Execução com Vitest e cobertura no pipeline CI

## Execução local
```bash
npm run test
npm run test:ci
```

## Critérios
- Tests verdes
- Cobertura mínima configurada no Vitest

## English Summary
Vitest runs unit tests for prediction logic and API client with coverage enforced in CI.
