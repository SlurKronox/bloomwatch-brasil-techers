# Arquitetura

## Visão geral
Aplicação web para monitoramento de florada com dados ambientais e previsão baseada em heurística.

## Camadas
- Interface: componentes React e dashboards
- Domínio: `PredictionService` e serviços auxiliares
- Integração: Supabase (dados e Edge Functions)

## Decisão de repositório
Este é o repositório oficial (primary source) da linha BloomWatch.

## English Summary
TypeScript-first architecture with UI, domain services, and Supabase integrations.
